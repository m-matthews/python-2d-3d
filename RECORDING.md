# Recording Settings

The following commands help to ensure the windows used for Video Recording are correctly set to 1280 x 756 pixels.


## Inkscape

```bash
xdotool search -name " - Inkscape" windowsize 1280 746
```


## Blender 3D

```bash
./blender -p 100 100 1280 756
```
